# order-inventory-microservices-assignment
Spring Boot Microservices Assignment implementing Order Service and Inventory Service with REST communication, H2 database, and Factory Design Pattern.
Setup
1.Clone the repository
git clone https://github.com/rraksha28/order-inventory-microservices-assignment.git
cd order-inventory-microservices-assignment

2.Build and Run Inventory Service:
cd inventory-service
mvn clean install
mvn spring-boot:run
Runs on port 8081

3.Build and Run order service
cd ../order-service
mvn clean install
mvn spring-boot:run
Runs on port 8080
